1. Motivation, Lernziele und Voraussetzungen

1.1 Motivation

Wie programmiere ich in JavaScript?

1.2 Lernziele

  • Ich kenne grundlegende Regeln zum Schreiben von JavaScript-Code.
  • Ich weiß, was Variablen in JavaScript sind.
  • Ich weiß, wie man eine Variable deklariert und ihr einen Wert zuweist.
  • Ich weiß, welche Datentypen es in JavaScript gibt.
  • Ich weiß, was "undefined", "nicht definiert", "null" und "NaN" in JavaScript bedeutet.
  • Ich weiß, was Operatoren, Operanden und Operationen in JavaScript sind und wie man sie benutzt.
  • Ich weiß, welche Kontrollstrukturen es in JavaScript gibt und wie man die wichtigsten davon einsetzt.

1.3 Voraussetzungen

  • Ich bin bereits mit HTML vertraut.
  • Ich weiß, was JavaScript ist und wofür man es einsetzt.
  • Ich weiß, wie man JavaScript in einer Webseite nutzt.
