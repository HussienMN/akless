1. Motivation, Lernziele und Voraussetzungen

Wie kann ich meine Webseite mit JavaScript anreichern? In dieser Lerneinheit lernen Sie die drei Möglichkeiten kennen, wie man JavaScript in einer Webseite benutzt und was die wichtigsten Parameter sind, die bei den Referenzen auf JavaScripts angeben werden können. Zudem erfahren Sie, was Sie tun können, wenn Anwender in ihrem Webbrowser JavaScript deaktiviert haben. Voraussetzung ist, dass sie bereits mit HTML vertraut sind und bereits wissen, was JavaScript ist und wofür man es einsetzt.

1.1 Lernziele

  • Ich kenne die drei Möglichkeiten, wie man JavaScript in einer Webseite benutzt.
  • Ich kenne die wichtigsten Parameter, die bei den Referenzen auf JavaScripts angeben werden können.
  • Ich weiß, wie auf Anwender mit deaktiviertem JavaScript im Webbrowser reagiert werden kann.

1.2 Voraussetzungen

  • Ich bin bereits mit HTML vertraut.
  • Ich weiß, was JavaScript ist und wofür man es einsetzt.

2. Inhalt

2.1 Skripte in Webseiten verankern und der gemeinsame Namensraum

Um JavaScript in einer Webseite zu verwenden, können Sie auf ein Inline-Skript, ein internes Skript oder die Verbindung mit einer externen Datei setzen. Sie können durchaus auch mehrere Skriptbereiche in einer Webseite verwenden. Alle JavaScripts in der Webseite landen in einem gemeinsamen Namensraum.

2.3 Kein JavaScript?

Die meisten Web-Anwender haben JavaScript in ihren Browsern aktiviert. Mit dem noscript-Tag können Sie Besuchern, die JavaScript deaktiviert haben, gezielt Informationen zur Verfügung stellen. Auch ein indirektes Verfahren lässt zwischen Browsern mit und ohne JavaScript unterscheiden.
