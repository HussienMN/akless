1. Motivation, Lernziele und Voraussetzungen

1.1 Motivation

Bei komplexeren Webseiten möchte man eine strikte Trennung zwischen Struktur und Funktionalität. Hierzu sollte die Ereignisbehandlung per JavaScript und nicht per HTML erfolgen.

1.2 Lernziele

  • Ich weiß, was Funktionsreferenzen, anonyme Funktionen und deren Sinn und Zweck ist.
  • Ich weiß, was ein Event-Objekt ist und wie man es verwertet.

1.3 Voraussetzungen

  • Ich bin bereits mit HTML vertraut.
  • Ich weiß, was JavaScript ist und wofür man es einsetzt.
  • Ich weiß, wie man JavaScript in einer Webseite nutzt.
  • Ich bin mit den wichtigsten Konzepten und der Syntax von JavaScript vertraut.
  • Ich weiß, wie man in JavaScript Funktionen einsetzt.
  • Ich bin mit Objekten in JavaScript vertraut.
  • Ich habe das DOM-Konzept verstanden und weiß wie man Elemente der Webseite selektiert.

2. Inhalt

2.1 Funktionsreferenzen und anonyme Funktionen

Funktionsreferenzen und anonyme Funktionen werden in der Regel an JavaScript-Eventhandler gekoppelt, die bei einem bestimmten Ereignis die referenzierte Funktion aufrufen. Sie können damit die Ereignisbehandlung vollkommen von HTML lösen.
